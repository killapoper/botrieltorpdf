# Краткая инструкция по запуску

## Быстрый старт

1. **Установите зависимости:**
```bash
cd "D:\desktop\Vse\Rielt bot pdf"
pip install -r requirements.txt
```

2. **Запустите все сервисы:**
Дважды кликните на файл `start_all.bat`

3. **Протестируйте работу:**
Откройте в браузере: `D:\desktop\Vse\Site rielt\test.html`

## Что происходит

- **Web Server** (порт 5000) - принимает заявки с сайта
- **Telegram Bot** - отправляет квиз и получает уведомления
- **Заявки с сайта** → Web Server → Telegram админам (ID: 1652676928, 327895912)

## Устранение проблем

**Заявки не приходят:**
- Проверьте, что оба сервиса запущены
- Проверьте консоль на ошибки
- Убедитесь, что в `.env` правильный токен бота

**CORS ошибка:**
- Убедитесь, что установлен `flask-cors`
- Перезапустите веб-сервер

**Бот не отвечает:**
- Проверьте токен в `.env`
- Убедитесь, что бот не заблокирован
