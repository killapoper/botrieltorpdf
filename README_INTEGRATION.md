# Интеграция сайта с Telegram ботом

## Что было сделано

Теперь заявки с сайта автоматически приходят в Telegram бот администраторам (ID: 1652676928 и 327895912).

## Структура проекта

```
Rielt bot pdf/
├── bot.py              # Основной Telegram бот (квиз)
├── web_server.py       # Веб-сервер для приема заявок с сайта
├── requirements.txt    # Зависимости Python
├── start_all.bat       # Скрипт для запуска всех сервисов
└── data/
    ├── leads.txt       # Заявки из бота (квиз)
    ├── site_leads.txt  # Заявки с сайта
    └── users.txt       # Пользователи бота

Site rielt/
├── index.html          # Главная страница сайта
└── script.js           # Обновлен для отправки заявок на сервер
```

## Как это работает

1. **Пользователь заполняет форму на сайте** (имя, телефон, тип услуги)
2. **JavaScript отправляет данные** на веб-сервер (http://localhost:5000/api/submit-form)
3. **Веб-сервер получает данные** и отправляет уведомление в Telegram обоим админам
4. **Админы получают сообщение** с пометкой "🌐 НОВАЯ ЗАЯВКА С САЙТА!"
5. **Данные сохраняются** в файл `data/site_leads.txt`

## Установка и запуск

### Вариант 1: Автоматический запуск (рекомендуется)

1. Откройте папку `D:\desktop\Vse\Rielt bot pdf`
2. Дважды кликните на файл `start_all.bat`
3. Откроются два окна:
   - Web Server (порт 5000)
   - Telegram Bot

### Вариант 2: Ручной запуск

1. Установите зависимости:
```bash
cd "D:\desktop\Vse\Rielt bot pdf"
pip install -r requirements.txt
```

2. Запустите веб-сервер (в первом терминале):
```bash
python web_server.py
```

3. Запустите Telegram бота (во втором терминале):
```bash
python bot.py
```

## Команды админа в боте

Отправьте `/admin` в Telegram боте, чтобы открыть панель администратора:

- **📊 Статистика** - общая статистика (пользователи, заявки из бота, заявки с сайта)
- **📋 Заявки из бота** - последние 10 заявок из квиза
- **🌐 Заявки с сайта** - последние 10 заявок с сайта
- **🔄 Обновить PDF** - загрузить новый PDF-файл для рассылки

## Устранение неполадок

### Ошибка "Failed to fetch"
- Убедитесь, что веб-сервер запущен (`python web_server.py`)
- Проверьте, что сервер работает на порту 5000

### Заявки не приходят в Telegram
- Проверьте, что бот запущен (`python bot.py`)
- Убедитесь, что в `.env` файле указан правильный `TELEGRAM_BOT_TOKEN`
- Проверьте ID админов в коде `web_server.py` (строка 20)

### CORS ошибки
- Убедитесь, что установлен `flask-cors`: `pip install flask-cors`

## Безопасность

⚠️ **ВАЖНО для продакшена:**
- Не храните токен бота в открытом виде
- Используйте HTTPS для веб-сервера
- Добавьте rate limiting для защиты от спама
- Валидируйте входящие данные
- Используйте переменные окружения для конфиденциальных данных
